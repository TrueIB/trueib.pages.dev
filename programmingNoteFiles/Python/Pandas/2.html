2. 两大数据结构

1. Series - 带标签的数组
可以把它理解为“一列数据”。 
import pandas as pd

# 创建一个 Series
s = pd.Series([90, 95, 88, 70], index=['小明', '小红', '小刚', '小丽'])
print(s)
输出： 
小明    90
小红    95
小刚    88
小丽    70
dtype: int64
左边是索引（index，可以自己指定，默认是 0,1,2,3...），右边是数据。它就像一本字典，可以通过名字（索引）找到对应的值。
2. DataFrame - 表格
这是 Pandas 的绝对主角，它是由多个 Series 拼成的 “一张表格”。 
# 创建一个 DataFrame
data = {
    '语文': [90, 95, 88, 70],
    '数学': [85, 92, 78, 98],
    '英语': [88, 90, 85, 80]
}
df = pd.DataFrame(data, index=['小明', '小红', '小刚', '小丽'])
print(df)
输出： 
     语文  数学  英语
小明  90  85  88
小红  95  92  90
小刚  88  78  85
小丽  70  98  80
  • columns：列名（语文、数学、英语）
  • index：行索引（小明、小红...）
  • 每个单元格里是具体的数据。
后面所有的操作都是围绕 DataFrame 进行的。
