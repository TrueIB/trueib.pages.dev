5. 挑选与切片

从表格里挑选数据的方法。

1. 选一列（得到一个 Series）
df['语文']        # 选择“语文”列
df.语文           # 同上，但列名是中文或有空格时不能用这种方法
2. 选多列（得到一个 DataFrame）
df[['语文', '数学']] # 注意！里面是两个中括号
3. 选一行（得到一个 Series）
df.loc['小明']     # 通过标签索引选择“小明”这一行
df.iloc[0]        # 通过位置索引选择第0行（第一行）
4. 选多行
df.loc[['小明', '小刚']] # 选择“小明”和“小刚”行
df.iloc[0:2]           # 选择第0行（第一行）到第1行（第二行）（左闭右开）
5. 同时选择行和列
df.loc['小明', '语文']           # 选中“小明”行，“语文”列交叉的那个单元格 -> 90
df.loc[['小明', '小刚'], ['语文', '英语']] # 选中一个子表格
df.iloc[0:2, 1:3]              # 用位置索引，选中第0-1行，第1-2列
