8. 计算与分组

1. 简单计算
Pandas 可以直接对一列进行数学运算。 
df['总分'] = df['语文'] + df['数学'] + df['英语'] # 新增一列“总分”
df['语文平均'] = df['语文'].mean()                # 新增一列，值都是语文平均分
2. 分组聚合 (Groupby)
“分组聚合” 就是 “按某个条件分组，然后对每组进行统计”。 | 问题：男生和女生的平均数学成绩分别是多少？ 
# 假设我们有一列“性别”
df.groupby('性别')['数学'].mean()
  • groupby('性别')：按“性别”分组（比如分成了“男”、“女”两组）
  • ['数学']：我关心的是“数学”这一列
  • .mean()：对分好的组里的数学成绩求平均值
或者一次性做多个统计： 
df.groupby('性别')['数学'].agg(['mean', 'count', 'max'])
# 输出：每个性别的数学平均分、人数、最高分
