3. 读取与写入

数据不能全靠手打，得从文件里读。

1. 读取数据
# 从 CSV 文件读取（最常用）
df = pd.read_csv('filename.csv')

# 从 Excel 文件读取
df = pd.read_excel('filename.xlsx', sheet_name='Sheet1')

# 从网页的表格读取
tables = pd.read_html('https://url.com/with-table.html')
df = tables[0] # 通常第一个表格就是我们想要的
2. 写入数据
# 写入 CSV 文件
df.to_csv('new_filename.csv', index=False) # index=False表示不保存行索引

# 写入 Excel 文件
df.to_excel('new_filename.xlsx', sheet_name='Data')
