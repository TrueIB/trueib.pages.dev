# TrueIB的个人网站项目
链接：[https://trueib.pages.dev](https://trueib.pages.dev, "访问个人网站")
 
## 🌟 功能特性
- ✅ CSS + JavaScript 切换主题
- ✅ 响应式布局（适配移动/桌面端）
- ✅ JavaScript setTimeout函数（侧边栏滑动过程）
- ✅ JavaScript绑定（侧边栏按钮滚动）
